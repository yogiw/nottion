## Tech Stack

- Next JS
- Slate JS & React Slate
- DND-Kit
- Tailwind

## Features

- Type "/" to show commands
  - Heading, paragraph, image, code, quote
- Text styling & shortcut
  - CMD + b → Bold
  - CMD + i → Italic
  - CMD + u → Underline
- Drag and drop
